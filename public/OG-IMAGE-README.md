# Open Graph Image Guide

## Required Specifications

Create an `og-image.png` file with the following specifications:

- **Dimensions**: 1200 x 630 pixels
- **Format**: PNG or JPG
- **File size**: < 300KB for fast loading
- **Location**: Place in `public/og-image.png`

## Content Recommendations

The image should include:

1. **Daily English Logo**: Your brand logo prominently displayed
2. **Tagline**: "Học tiếng Anh mỗi ngày" or similar Vietnamese tagline
3. **Visual Elements**: 
   - English learning icons (books, speech bubbles, etc.)
   - Modern, clean design
   - Brand colors (#4F46E5 primary, #10B981 secondary)
4. **Background**: Professional gradient or solid color

## Design Tools

You can create this image using:

- **Canva**: https://www.canva.com/ (has social media templates)
- **Figma**: https://www.figma.com/
- **Adobe Photoshop/Illustrator**
- **Online OG Image Generators**: https://www.opengraph.xyz/

## Testing

After creating the image, test it with:

- **Facebook Sharing Debugger**: https://developers.facebook.com/tools/debug/
- **Twitter Card Validator**: https://cards-dev.twitter.com/validator
- **LinkedIn Post Inspector**: https://www.linkedin.com/post-inspector/
