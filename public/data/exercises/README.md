# Exercise Data Structure

This directory contains all exercise data organized by difficulty level and category.

## Structure

```
exercises/
├── beginner/
│   ├── daily-life.json
│   ├── education-work.json
│   ├── culture-arts.json
│   ├── health-wellness.json
│   ├── science-environment.json

│   ├── travel-transportation.json
│   └── philosophy-beliefs.json
├── intermediate/
│   └── (same categories)
└── advanced/
    └── (same categories)
```

## Categories

1. `daily-life` - Everyday activities and routines
2. `education-work` - School, career, and professional topics
3. `culture-arts` - Music, literature, traditions, and cultural practices
4. `health-wellness` - Physical and mental health topics
5. `science-environment` - Technology, nature, and environmental issues
6. `society-services` - Community, public services, and social topics
7. `travel-transportation` - Travel experiences and transportation
8. `philosophy-beliefs` - Ethics, religion, history, and philosophical topics

## Benefits of This Structure

- **Easy to manage**: Each file contains only 3-5 exercises
- **Better organization**: Clear separation by level and category
- **Scalable**: Easy to add new exercises or categories
- **Maintainable**: Easier to review and update specific sections
- **Version control friendly**: Smaller diffs when making changes
