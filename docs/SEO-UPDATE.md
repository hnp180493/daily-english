## Cải tiến kỹ thuật SEO

### 1. Tối ưu hóa Performance
- **Gom exercises vào 1 file**: Giảm từ 24 HTTP requests xuống còn 1 request duy nhất
- **Lazy loading**: Tất cả routes đều sử dụng lazy loading
- **Preload strategy**: Các trang quan trọng được đánh dấu `preload: 'high'`

### 2. Structured Data
- WebSite schema với SearchAction
- EducationalOrganization schema
- LearningResource schema cho exercises
- Breadcrumb navigation

### 3. Vietnamese SEO
- Hreflang tags cho tiếng Việt
- Vietnamese-specific keywords
- FAQs bằng tiếng Việt
- Breadcrumbs navigation

### 4. Meta Tags
- Open Graph tags cho social sharing
- Twitter Card tags
- Canonical URLs
- Robots meta tags

### Performance Optimization
```typescript
// Before: 24 requests
loadExercises() {
  const requests = levels.flatMap(level =>
    categories.map(category =>
      this.http.get(`/data/exercises/${level}/${category}.json`)
    )
  );
  combineLatest(requests).subscribe(...);
}

// After: 1 request
loadExercises() {
  this.http.get('/data/exercises/all-exercises.json').subscribe(...);
}
```

### Caching Strategy
```typescript
// Prevent duplicate requests
async getUserCurrentPath(): Promise<LearningPath | null> {
  if (this.loadingPromise) {
    return this.loadingPromise;
  }
  this.loadingPromise = this.loadUserCurrentPathInternal();
  try {
    return await this.loadingPromise;
  } finally {
    this.loadingPromise = null;
  }
}
```
